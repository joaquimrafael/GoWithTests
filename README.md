# Go With Tests

This repository contains examples and exercises for learning Go through test-driven development.

## Getting Started

Follow the steps below to set up and run the examples:

1. Clone the repository:

    ```bash
    git clone https://github.com/yourusername/GoWithTests.git
    ```

2. Navigate to the project directory:

    ```bash
    cd GoWithTests
    ```

3. Run the tests:

    ```bash
    go test ./...
    ```

## Structure

The repository is organized into the following sections:

- **Basics**: Fundamental Go concepts.
- **Intermediate**: More advanced topics and patterns.
- **Exercises**: Practice problems to reinforce learning.

## Basic Cycle

- Write a test
- Make the compiler pass
- Run the test, see that it fails and check the error message is meaningful
- Write enough code to make the test pass
- Refactor
